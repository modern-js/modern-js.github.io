Template strings

Template strings can contain executable expressions. Expressions are resolved at the point of definition.

Properties Operators
  const abc = 'abcdefg';
  const hello = 'hello world';

  abc.length // 3
  hello.length // 11

  // get a character
  abc[0] // "a"
  abc[100] // undefined

  hello === "hello world" // true
  hello > abc // true
